# 手机 App 安装与分享指南

> **本应用已部署上线，直接分享这个网址即可：https://fanque123.github.io/math-practice-app/**
> （源码仓库：https://github.com/fanque123/math-practice-app）

口算机器人是一个 PWA（网页应用），**不用上架任何应用商店**：朋友用手机浏览器打开上面的网址后「添加到主屏幕」，桌面就会生成 App 图标，点开全屏使用，断网也能玩。

## 第一步：把应用放到网上（只需做一次）

手机安装要求网址必须是 **HTTPS**（或本机 localhost）。任选一个免费静态托管，把整个项目文件夹传上去即可：

| 平台 | 大致步骤 |
| --- | --- |
| GitHub Pages | 新建仓库 → 上传全部文件 → Settings → Pages → 选 main 分支 → 得到 `https://你的用户名.github.io/仓库名/` |
| Cloudflare Pages | dash.cloudflare.com → Pages → 上传文件夹（Direct Upload）→ 得到 `https://xxx.pages.dev` |
| Netlify / Vercel | 官网注册 → 拖拽文件夹上传 → 自动分配 HTTPS 网址 |

上传时这些文件一个都不能少：`index.html`、`app.js`、`styles.css`、`manifest.json`、`service-worker.js`、`icon-192.png`、`icon-512.png`、`apple-touch-icon.png`。

> 以后改了代码，重新上传后把 `service-worker.js` 里的缓存版本号（如 `math-practice-v15`）加一，朋友手机才会自动更新。

## 第二步：朋友安装（把网址发给他们）

- **安卓手机**：用 Chrome / Edge 打开网址 → 点页面上的「📲 安装App」按钮，或浏览器菜单 →「安装应用 / 添加到主屏幕」。
- **iPhone / iPad**：用 **Safari** 打开网址 → 点底部「分享」按钮 →「添加到主屏幕」（页面上点「📲 安装App」也会弹出这个指引）。

安装后桌面出现「口算机器人」图标，点开就是全屏 App，离线可玩。

## 附：同一 WiFi 下快速试玩（不安装）

1. 电脑上双击 `start-server.bat` 启动服务，窗口会打印类似 `http://192.168.1.5:8000/` 的地址。
2. 手机连同一个 WiFi，浏览器打开这个地址即可直接玩。
3. 注意：这种方式是电脑临时分享，电脑关机就失效；且局域网地址不是 HTTPS，不能「安装」，只适合试玩。
