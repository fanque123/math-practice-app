# 手机 App 安装与分享指南

> **两种方式任选：**
> **① APK 安装包（推荐）**：手机直接打开下载链接 **https://github.com/fanque123/math-practice-app/releases/download/v1.0/math-practice-v1.0.apk** ，下载后点开即装，不经过浏览器运行，离线可用；也可以找作者要 `口算机器人.apk` 文件传输安装。
> **② 网页版（PWA）**：分享网址 https://fanque123.github.io/math-practice-app/ ，手机浏览器打开后「添加到主屏幕」。
> （源码仓库：https://github.com/fanque123/math-practice-app）

## 方式一：APK 安装包（推荐）

### 朋友怎么装

1. 手机浏览器打开上面的 APK 下载链接，下载完成点安装；或把 `口算机器人.apk` 文件传给手机再点开（QQ/数据线/网盘都行；**微信直发 APK 会被改名成 `.apk.1` 点不开**，需用文件管理器重命名回 `.apk`，或先打成 zip 压缩包再发）。
2. 安装时提示「禁止安装未知来源应用」→ 点「设置」→ 允许当前应用安装（装完可以关回去）。
3. 小米/华为等会弹「安全检测」提示，这是未上架商店的正常提醒，点「继续安装」即可。
4. 装完桌面出现「口算机器人」图标，点开即用，**不需要联网、不申请任何权限**。

### APK 版与网页版的差别

- 语音朗读（读题/鼓励）正常，用的是手机系统的语音引擎（小米一般是讯飞/小爱）。
- 语音答题（v1.1 起）正常：首次开始练习时会弹麦克风权限申请，允许后即可；之后也能在系统设置 → 应用管理 → 口算机器人 → 权限里开关。
- 内容是打包时的版本，之后网页端有更新不会自动同步，找作者要新 APK 覆盖安装即可。

## 方式二：网页版（PWA）

网址已部署好，直接分享 **https://fanque123.github.io/math-practice-app/** 即可，朋友按下面步骤安装。

### 朋友怎么安装（把网址发给他们）

- **安卓手机**：用 Chrome / Edge 打开网址 → 点页面上的「📲 安装App」按钮，或浏览器菜单 →「安装应用 / 添加到主屏幕」。
- **iPhone / iPad**：用 **Safari** 打开网址 → 点底部「分享」按钮 →「添加到主屏幕」（页面上点「📲 安装App」也会弹出这个指引）。

安装后桌面出现「口算机器人」图标，点开就是全屏 App，离线可玩。

## 附：同一 WiFi 下快速试玩（不安装）

1. 电脑上双击 `start-server.bat` 启动服务，窗口会打印类似 `http://192.168.1.5:8000/` 的地址。
2. 手机连同一个 WiFi，浏览器打开这个地址即可直接玩。
3. 注意：这种方式是电脑临时分享，电脑关机就失效；且局域网地址不是 HTTPS，不能「安装」，只适合试玩。
