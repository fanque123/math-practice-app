# 口算小助手

适合小学生的四则运算口算练习应用，支持手机 / 电脑浏览器。

## 功能

- 四大题型：乘法、加法、减法、除法；乘加减各 5 个难度，除法 4 个难度（全部整除）
- 两种陪伴角色可切换：擎天柱（答对喊「嘟嘟，好棒！」）/ 公主（鲜花背景，答对说「真棒！」）
- 「设置」面板：默认模式、按模式自定义答对/答错台词、每次默认题数（默认 30），自动保存
- 可设置每次题目数量（1~50），支持回车键直接开始练习
- 出题规则：乘法不含 1、加法必进位、减法必退位、除法必整除
- 自动朗读题目
- 读完题目后自动开始语音识别，无需点击按钮
- 识别失败可点击「重说」或手动输入答案
- 练习结束显示得分率（百分比），并按成绩说不同的鼓励语，附错题回顾
- 支持 PWA，可作为手机 App 安装（不用上架商店），详见 `安装分享指南.md`

## 推荐运行方式（避免每次弹麦克风权限）

直接双击 HTML 文件时，浏览器会把它当作 `file://` 协议，每次使用麦克风都需要重新授权。**推荐用本地服务器打开，权限只需同意一次。**

### 方法一：一键启动（Windows）

1. 双击 `start-server.bat`
2. 浏览器会自动打开 `http://localhost:8000`
3. 第一次使用时允许麦克风权限，之后整个练习过程都不再弹窗

### 方法二：命令行启动

```bash
node server.js
```

然后打开 `http://localhost:8000`。

## 装到手机上当 App 用（可分享给朋友）

**方式一 · APK 直装（推荐）**：手机浏览器打开 https://github.com/fanque123/math-practice-app/releases/latest/download/math-practice.apk 直接下载安装，离线可用（朗读和语音答题都正常，仅需麦克风权限）。
**方式二 · 网页版**：https://fanque123.github.io/math-practice-app/ —— 手机浏览器打开后「添加到主屏幕」。详细步骤和传文件避坑看 `安装分享指南.md`。同一 WiFi 下也可以直接手机访问启动时打印的局域网地址快速试玩。

## 直接使用（可能每次都要点麦克风权限）

如果只是想快速试试，也可以直接双击 `index.html`。但语音识别和权限体验不如本地服务器稳定。

## 技术说明

- 语音朗读：浏览器 Web Speech API，优先选择 Edge/Chrome 的中文自然语音（如 Microsoft Xiaoxiao Online）
- 语音识别：浏览器 Web Speech API，支持中文数字（一、十、百、千）和阿拉伯数字识别
- 题目生成：纯前端随机生成，无需网络
